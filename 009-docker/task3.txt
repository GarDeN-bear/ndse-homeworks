4. Подключитесь к контейнеру `first_node` с помощью exec и создайте текстовый файл любого содержания в `/var/first/data`

vboxuser@Ubuntu-20:~/netology/ndse-homeworks/009-docker$ docker exec -it first_node sh -c "echo 'Hello from first_node\!' > /var/first/data/file1.txt"

5. Добавьте еще один файл в папку `data` на хостовой машине

vboxuser@Ubuntu-20:~/netology/ndse-homeworks/009-docker$ echo 'Hello from host\!' > /home/data/file_host.txt

vboxuser@Ubuntu-20:~/netology/ndse-homeworks/009-docker$ docker stop first_node
